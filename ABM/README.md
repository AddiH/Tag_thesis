# ABM

A Mesa-based agent-based model of tag, used to test whether self-handicapping alone can keep a game of tag going.

![Chase dynamics](chase-dynamics.gif)

**The full model description is in the [ODD-protocol report](tag_abm_ODD_report.md).**

- `tag_abm.py` — the model (agents, rules, chase dynamics)
- `tag_abm_main.ipynb` — runs the parameter sweep, calibration, and figures
- `tag_abm_ODD_report.md` — full ODD description of the model
- `chase-dynamics.html` — [interactive chase visualisation](https://htmlpreview.github.io/?https://github.com/AddiH/Tag_thesis/blob/main/ABM/chase-dynamics.html)
